# Aikido Kankouji-ryu | 合気道 冠光寺流

Static website for the Aikido Kankouji-ryu dojo, built with [Astro](https://astro.build).

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) to preview.

## Build

```bash
npm run build
```

Static files are output to `dist/`.

## Deployment

The site deploys automatically to GitHub Pages on every push to `main` via GitHub Actions.

**Live URL:** https://hoshiko-tech.github.io/aikido-kankouji-ryu/

### Manual deployment setup

1. Go to repository **Settings > Pages**
2. Set source to **GitHub Actions**
3. Push to `main` to trigger a build
